## Flex 

### justify-content、align-items

```css
.outer {
  display: flex;
  justify-content: center; /* 水平居中 */
  align-items: center; /* 垂直居中 */
}
```

### flex + margin

```css
.outer {
  display: flex;
}

.inner {
  margin: auto;
}
```

### 兼容性

IE 10+, Chrome 4+, Firefox 2+

Android 2.3+, iOS 6+

---

<span id="jump2"></span>

## Grid

### justify-self、align-self

```css
.outer {
  display: grid;
}

.inner {
  justify-self: center; /* 水平居中 */
  align-self: center; /* 垂直居中 */
}
```

### grid + margin

```css
.outer {
  display: grid;
}

.inner {
  margin: auto;
}
```

### 兼容性

IE 10+, Chrome 57+, Firefox 52+

Android 6+, iOS 10.3+

<span id="jump3"></span>

## absolute + transform

### 实现

```css
.outer {
  position: relative;
}

.inner {
  position: absolute;
  left: 50%;
  top: 50%;
  transform:translateX(-50%); // 水平居中
  transform: translateY(-50%); // 垂直居中
}
```

### 兼容性

IE 9+, Chrome 4+, Firefox 3.5+

Android 3+, iOS 6+

---

<span id="jump4"></span>

## absolute + calc

### 实现

```css
.outer {
  position: relative;
}

.inner {
  width: 100px;
  height: 100px;
  position: absolute;
  left: calc(50% - 50px); // 水平居中
  top: calc(50% - 50px); // 垂直居中
}
```

缺点：需要固定居中元素的宽高

### 兼容性

IE 9+, Chrome 19+, Firefox 4+

Android 4.4+, iOS 6+

---

<span id="jump5"></span>

## absolute + 负margin

### 实现

```css
.outer {
  position: relative;
}

.inner {
  width: 100px;
  height: 100px;
  position: absolute;
  left: 50%;
  top: 50%;
  margin-left: -50px; // 水平居中
  margin-top: -50px; // 垂直居中
}
```

缺点：需要固定居中元素的宽高

### 兼容性

IE 6+, Chrome 4+, Firefox 2+

Android 2.3+, iOS 6+

---

<span id="jump6"></span>

## absolute + margin: auto

### 实现

```css
.outer {
  position: relative;
}

.inner {
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
}
```

缺点：如果有需要，居中元素的宽高需要给定，否则其宽高会被设为100%

### 兼容性

IE 6+, Chrome 4+, Firefox 2+

Android 2.3+, iOS 6+

<span id="jump7"></span>

## writing-mode

该属性不仅可以使文字显示方向改变，还能使所有作用于水平方向上的属性，变成作用于垂直方向

所以，通过 writing-mode 和 text-align: center 就可以实现垂直居中了

### 实现

DOM结构：

```css
<div class="outer">
  <div class="inner">
    <div class="content"></div>
  </div>
</div>
```

CSS:

```css
.outer {
  writing-mode: vertical-lr;
  text-align: center;
}

.inner {
  display: inline-block;
  width: 100%;
  height: auto;
  writing-mode: horizontal-tb;
  text-align: center;
}

.content {
  display: inline-block;
  text-align: left; /* 重置文字位置（如果需要） */
}
```

缺点：需要修改 DOM 结构（为了居中元素，需要嵌套两层父元素）

优点：不需要固定居中元素的宽高

### 兼容性

IE 6+, Chrome 4+, Firefox 3.5+

Android 2.3+, iOS 5.1+

## 等值padding或margin

给父元素添加等值的上下（垂直居中）或左右（水平居中）padding或margin

这时候父元素是不能设置高度的，要让它自动被填充起来

---

<span id="jump9"></span>

## margin: auto

```css
.inner {
  display: block; /* 还可以是 table | flex | grid，但不能是 inline-xxx */
  margin: 0 auto;
}
```

缺点：需要固定居中元素的宽

---

<span id="jump10"></span>

## text-align + inline-block

```css
.outer {
  text-align: center;
}

.inner {
  display: inline-block;
  text-align: left; /* 重置文字位置（如果需要） */
}
```

缺点：为了居中元素，使文本也居中了，因此可能需要重置文本位置

优点：不需要固定居中元素的宽

---

<span id="jump11"></span>

## height + line-height

设置line-height等于height

只能使文本内容垂直居中，不能使容器居中

---

<span id="jump12"></span>

## table-cell + vertical-align

```css
.outer {
  display: table-cell;
  vertical-align: middle;
}
```

---

<span id="jump13"></span>

## inline-block + line-height

```css
.outer {
  line-height: 300px;
}

.inner {
  line-height: initial; /* 重置 */
  vertical-align: middle;
  display: inline-block;
}
```

缺点：需要知道其父元素高度

---

<span id="jump14"></span>

## 伪元素 + vertical-align

```css
.outer::after {
  content: "";
  display: inline-block;
  vertical-align: middle;
  height: 100%;
}

.inner {
  display: inline-block;
  vertical-align: middle;
}
```

---

<span id="jump15"></span>

## text-align无效

text-align只能作用于inline元素，因此为block元素设置将无法起到剧中效果

---

<span id="jump16"></span>

## margin: auto无效

- 没有给要设定的元素指定宽度

- 元素的display属性不为“block”，“inline-block”属性同样无效

- 该元素有浮动
